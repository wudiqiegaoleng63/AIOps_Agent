# 第 8 章 前端与联调

## 8.1 启动流程

```bash
# 1. 启动 Milvus
docker compose -f vector-database.yml up -d

# 2. 启动 MCP 服务器（两个终端）
python mcp_servers/cls_server.py      # 端口 8003
python mcp_servers/monitor_server.py  # 端口 8004

# 3. 启动 FastAPI
uvicorn app.main:app --host 0.0.0.0 --port 9900 --reload

# 4. 上传知识库文档
curl -X POST http://localhost:9900/api/upload -F "file=@aiops-docs/cpu_high_usage.md"
# ... 对每个文档重复
```

或使用一键命令：
```bash
make init   # Docker + 服务 + 文档上传
```

Windows 环境使用批处理脚本：
```powershell
.\start-windows.bat
```

## 8.2 SSE 事件流对接

前端通过 `EventSource` 或 `fetch` 消费 SSE：

### RAG 流式对话

```javascript
const response = await fetch('/api/chat_stream', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({Id: sessionId, Question: question})
});

const reader = response.body.getReader();
const decoder = new TextDecoder();

while (true) {
    const {done, value} = await reader.read();
    if (done) break;
    const text = decoder.decode(value);
    // 解析 SSE 格式: "data: {json}\n\n"
    const data = JSON.parse(text.replace('data: ', ''));
    if (data.type === 'content') appendToChat(data.data);
    if (data.type === 'done') break;
}
```

// AIOps 诊断事件类型:
