## 2.2 配置管理

使用 Pydantic Settings 实现类型安全的配置，从 `.env` 文件加载：

```python
# app/config.py
from typing import Dict, Any
from pydantic_settings import BaseSettings, SettingsConfigDict


class Settings(BaseSettings):
    model_config = SettingsConfigDict(
        env_file=".env",
        env_file_encoding="utf-8",
        case_sensitive=False,  # 环境变量不区分大小写
        extra="ignore",        # 忽略多余的环境变量
    )

    # 应用配置
    app_name: str = "SuperBizAgent"
    port: int = 9900
    debug: bool = False

    # DashScope 配置
    dashscope_api_key: str = ""
    dashscope_model: str = "qwen-max"
    dashscope_embedding_model: str = "text-embedding-v4"

    # Milvus 配置
    milvus_host: str = "localhost"
    milvus_port: int = 19530

    # RAG 配置
    rag_top_k: int = 3
    rag_model: str = "qwen-max"
    chunk_max_size: int = 800
    chunk_overlap: int = 100

    # MCP 服务配置
    mcp_cls_url: str = "http://localhost:8003/mcp"
    mcp_monitor_url: str = "http://localhost:8004/mcp"

    @property
    def mcp_servers(self) -> Dict[str, Dict[str, Any]]:
        """聚合 MCP 服务器配置"""
        return {
            "cls": {"transport": "streamable-http", "url": self.mcp_cls_url},
            "monitor": {"transport": "streamable-http", "url": self.mcp_monitor_url},
        }


config = Settings()  # 全局单例
```

**要点**：
- `case_sensitive=False` 让环境变量名和代码字段名可以不同大小写（`DASHSCOPE_API_KEY` → `config.dashscope_api_key`）
- `mcp_servers` 用 `@property` 聚合，供 MCP 客户端直接使用
- 全局单例 `config`，其他模块直接 `from app.config import config`

## 2.3 FastAPI 应用入口

```python
# app/main.py
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware
from contextlib import asynccontextmanager

from app.config import config
from app.core.milvus_client import milvus_manager
from app.api import chat, health, file, aiops


@asynccontextmanager
async def lifespan(app: FastAPI):
    """应用生命周期管理"""
    # 启动时：连接 Milvus
    milvus_manager.connect()
    yield
    # 关闭时：断开 Milvus
    milvus_manager.close()


app = FastAPI(
    title=config.app_name,
    lifespan=lifespan,
)

app.add_middleware(
    CORSMiddleware,
    allow_origins=["*"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

# 注册路由
app.include_router(health.router, tags=["健康检查"])
app.include_router(chat.router, prefix="/api", tags=["对话"])
app.include_router(file.router, prefix="/api", tags=["文件管理"])
app.include_router(aiops.router, prefix="/api", tags=["AIOps"])
```

**要点**：
- 使用 `lifespan` 而非 `on_event`，这是 FastAPI 推荐的生命周期管理方式
- Milvus 连接放在 `lifespan` 里而非模块导入时，确保应用关闭时能正确释放
- 路由按功能分组，chat/aiops/file/health 各自独立

## 2.4 日志配置

使用 Loguru，在 `app/__init__.py` 中配置：

```python
# app/__init__.py
from loguru import logger
import sys

logger.add(
    "logs/app_{time:YYYY-MM-DD}.log",
    rotation="00:00",     # 每天轮转
    retention="30 days",  # 保留 30 天
    level="INFO",
    encoding="utf-8",
)
```

所有模块直接 `from loguru import logger` 即可使用。
